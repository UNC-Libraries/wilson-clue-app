# Clue
A Laravel / MySQL game management application and player interface for the biannual, live-action Clue game at Wilson Library.

## Installation
See https://gitlab.lib.unc.edu/cappdev/clue-vm for local development set up

### Note
Any emails write to the log

## Asset Compilation
1. Watch for changes to assets in development and for production:
   ```bash
   $ cd $CLUE_HOME
   $ npm run build
   ```
